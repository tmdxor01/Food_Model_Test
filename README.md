# 졸업프로젝트

## 안드로이드 음식 인식 모델 성능 테스트

Android Studio 환경에서 음식 이미지 인식 모델의 성능을 테스트하기 위한 졸업프로젝트이다.  
모델의 예측 결과를 기반으로 정확도(Accuracy)를 확인하여 모델 성능 개선과 학습 최적화를 목표로 한다.

## 개발 환경
- Android Studio
- Kotlin

## 주요 코드
- app/src/main/java/com/example/myapplication/MainActivity.kt

## 기능
- 음식 이미지 인식
- 모델 예측 결과 출력
- 정확도(Accuracy) 확인
