# 졸업프로젝트

## 안드로이드 음식 인식 모델 성능 테스트

졸업프로젝트 FoodMate를 위해 Android Studio 환경에서 온디바이스 음식 인식 모델 성능을 테스트하고 학습 방식을 결정하기 위한 간단한 앱이다.
모델의 예측 결과를 기반으로 정확도(Accuracy)를 확인하여 모델 성능 개선과 학습 최적화를 목표로 한다.
기본 기능도 추가하여 FoodMate의 기본이 되는 앱이다.

## 개발 환경
- Android Studio
- Kotlin

## 주요 코드
- app/src/main/java/com/example/myapplication/MainActivity.kt

## 기능
- 음식 이미지 인식
- 모델 예측 결과 출력
- 정확도(Accuracy) 확인
